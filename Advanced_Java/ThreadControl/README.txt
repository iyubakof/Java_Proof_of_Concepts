This is a program demostrates Thread Control with 5 threads.
The concepts utilized are:
	KeyListener
	JFrame
	Threads
	Passing Objects

Instructions:
1. All threads will execute once the program starts
2. At anytime, press an integer between 1 and 5 to take control of a thread.
3. Press S to stop a thread you have control over
4. Press T, N, or L to set the priority of a thread you have control over.
